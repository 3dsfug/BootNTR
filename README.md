# BootNTR

Boots NTR CFW.

# Compatibility

This is compatible with 11.0 and 11.1 firmware on all devices. 

# Future updates

If a future update breaks it, please open an issue stating your console, firmware version, and attach _all_ dumps on the SD card root.
